<h2>Getting Started</h2>
<div>
<p>is project was built using React,Redux, Material UI, CSS, JavaScript,HTML. An Ecommerce website for electronics buying and selling and for running on your local environment you should follow these guidelines</p>
<h3>Prerequisites</h3>
<ul>
<li>React</li>
<li>Redux</li>
<li>MongoDb</li>
<li>Express</li>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>
<h3>Setup</h3>
```
git clone https://github.com/rozodkarbharat/full-stack-ecommerce.git
```
<h3>Install</h3>
<p>To install all the dependences of the project, run the following command:</P>
```
npm install
```
<p>To run the application got to the client folder and run the following command:</P>
```
npm run start
```
